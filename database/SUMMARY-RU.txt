🚀 БЫСТРЫЙ СТАРТ:

# 1. Установить зависимости
cd database
npm install

# 2. Создать .env
cp .env.example .env

# 3. Запустить Docker (из корня)
cd ..
docker-compose up -d

# 4. Применить миграции
make migrate

# 5. Загрузить дамп с автомобилями
make load-dump

# 6. Загрузить тестовых пользователей
make seed

# 7. Проверить
docker exec -it cars_postgres psql -U postgres search_db
